🧠 HireMind AI 🤖

Smart AI Career Assistant

HireMind AI is a simple and intelligent web application that helps users improve their resumes, practice interviews, and plan their learning journey using artificial intelligence. It is designed to make career preparation easier, smarter, and more personalized.



✨ What HireMind AI Does

📝 AI Resume Analyzer
Checks resumes and gives feedback to improve quality and ATS score.

🎯 Interview Practice with AI
Generates technical and behavioral questions and helps users prepare for interviews.

🧭 Personalized Learning Roadmap
Creates step-by-step learning paths based on skills and career goals.

📊 All-in-One Dashboard
Manage resumes, interviews, and learning plans in one place.

📱 Responsive Design
Works smoothly on desktop and mobile devices.



🛠 Tech Stack 💻

⚛️ React & TypeScript

🌐 JavaScript, HTML, CSS

🤖 Google Gemini API

🔗 REST APIs

🧩 Modular Components

🧑‍💻 Git & GitHub



🏗 How It Works 🧠

HireMind AI connects the frontend with AI services through APIs.
It processes user input, interacts with the AI model, and shows smart results in a clean and user-friendly interface.

🚀 Getting Started
# Clone the repository
git clone https://github.com/your-username/HireMind-AI.git

# Go to the project folder
cd HireMind-AI

# Install dependencies
npm install

# Start the app
npm run dev





🔑 Setup AI Key 🤖

Create a .env file and add your Gemini API key:

VITE_GEMINI_API_KEY=your_api_key_here
